os：用来获取操作系统（windows）的信息，如文件的路径，名称，大小
for example:
os.getcwd() : 获取当前工作目录
os.chdir() : 改变当前工作目录
os.listdir() : 获取指定目录的文件列表
os.mkdir() : 创建目录
os.system() ： 执行系统命令
os.path.abspath() : 获取绝对路径

sys:获取解释器的变量（pycharm）
sys.getdefaultencoding() : 获取默认编码
sys.thread_info() :获取线程信息
sys.stdin :标准输入流
sys.stdout :标准输出流
